# stada
* stada - Standard Data
* Standard ISO codes for country data
* Query country data in GraphQL mode


## Installation
1. Clone the repository
2. Optional: Create a .env file in the root folder and specify a port ```PORT = 3000```
3. Install using npm 
```
npm i
```

## Usage
Run in debug mode 
```
npm run dev
```

Transpile
```
npm run build
```
